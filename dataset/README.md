These are the datasets used to train the models in the ``models/`` folder. 

Some of these were synthetically generated with NumPy and others from real sources, while others (designated with the prefix ``sampled`` are subsets of others indicated here)

---

### Dataset directory

|          Dataset         |   Generated By / From   |    Model(s)         | Features |
| :-----------------------: | :--------------: | :------------------------:|:--------:|
| ``sampled_flood_data.csv`` | ``flood_yes.csv``, ``flood_no.csv`` | FV-FloodNet | ``rainfall``	``water_level``	``elevation``	``slope``	``distance_from_river`` |
|     ``no_fire.csv``      |      NumPy       |   FireNet  |  ``temperature`` ``humidity`` ``wind_speed`` ``vegetation_index`` ``elevation`` |
|  ``high_temp_fire.csv``  |      NumPy       |   FireNet  |  ``temperature`` ``humidity`` ``wind_speed`` ``vegetation_index`` ``elevation`` |
|    ``flood_yes.csv``     |      NumPy       |     FV-FloodTrustNet     | ``rainfall``	``water_level``	``elevation``	``slope``	``distance_from_river`` |
|     ``flood_no.csv``     |      NumPy       |     FV-FloodTrustNet     | ``rainfall``	``water_level``	``elevation``	``slope``	``distance_from_river`` |
| ``final_fire_dataset2.csv`` |  NumPy, [FIRMS](https://firms.modaps.eosdis.nasa.gov/data/active_fire/modis-c6.1/csv/MODIS_C6_1_Global_7d.csv) | FireNet, FireTrustNet |  ``temperature`` ``humidity`` ``wind_speed`` ``vegetation_index`` ``elevation`` |
| ``pluvial_flood_data_balanced.csv`` | NumPy | PV-FloodNet, PV-FloodTrustNet | ``rainfall_intensity``	``impervious_ratio``	``drainage_density``	``urbanization_index``	convergence_index`` |
| ``flash_flood_data.csv`` | NumPy | FlashFloodNet, FlashFloodTrustNet | ``rainfall_intensity``	``slope``	``drainage_density``	``soil_saturation``	``convergence_index`` |
| ``earthquake_data.csv`` | NumPy | QuakeNet, QuakeTrustNet | ``seismic_moment_rate``	``surface_displacement_rate``	``coulomb_stress_change``	``average_focal_depth``	fault_slip_rate`` | 
| ``hurricane_data.csv`` | NumPy | HurricaneNet, HurricaneTrustNet | ``sea_surface_temperature``	``ocean_heat_content``	``mid_level_humidity``	``vertical_wind_shear``	potential_vorticity`` |
| ``tornado_data.csv`` | NumPy | TornadoNet, TornadoTrustNet | ``storm_relative_helicity``	``CAPE``	``lifted_condensation_level``	``bulk_wind_shear``	``significant_tornado_param`` |
